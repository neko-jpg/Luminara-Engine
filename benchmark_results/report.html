Methodology & Reproducibility

Benchmark Descriptions

  • BunnyMark: Classic sprite rendering stress test. Spawns sprites incrementally until FPS drops below 60. Measures maximum sprite count at 60 FPS.
  • ECS Iteration: Tests entity component system performance. Creates large numbers of entities with multiple components and measures iteration throughput.
  • Startup Time: Measures time from process start to first rendered frame for both empty and complex projects.
  • Rendering Stress: Tests rendering pipeline with complex scenes containing many entities, lights, and materials.
  • Physics: Tests physics simulation performance with many active rigid bodies and collision detection.

Comparison Baselines

Performance comparisons are made against published benchmarks from:

  • Bevy: Rust game engine with ECS architecture
  • Godot: Popular open-source game engine
  • Unity: Industry-standard game engine (where applicable)

Reproducibility

To reproduce these benchmarks:

  1. Clone the Luminara Engine repository
  2. Run: cargo run --release --example [benchmark_name]_benchmark
  3. Results will be saved to benchmark_results/

Continuous Tracking

Benchmarks are run automatically on every major release and tracked over time to ensure performance improvements and catch regressions early.
